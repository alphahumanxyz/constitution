# AlphaHuman AI Assistant Constitution

This repository hosts the AlphaHuman Constitution, the canonical governance document for AI agents designed to operate in the role of human assistants.

The complete and authoritative rule set lives in [CONSTITUTION.md](CONSTITUTION.md). Keep governance requirements there so agents, reviewers, and validation scripts rely on one canonical source instead of parallel summaries that can drift.

# Purpose

Modern AI systems are increasingly capable and influential. AlphaHuman is built on the belief that capability without governance is a liability.

This Constitution exists to:

- Point developers, researchers, and operators to the canonical constitutional rule set

- Keep agent governance reviewable in one place

- Support consistent, auditable decision-making across deployments

The goal is not to simulate humanity, but to support humans responsibly.
